RuneShelf

Product Overview

RuneShelf is an iPhone companion app for Riftbound players focused on collection tracking, deck building, deck sharing, friends, and manual match support. It is not a standalone digital gameplay client.

Product summary

RuneShelf helps players organize their card collection, build decks, track owned and wanted cards, save deck versions, manage match history, browse friends, and inspect card information in a mobile-friendly interface.

Platform

iPhone / iOS

Primary use case

Deck builder and card library

Audience

Riftbound players and community members

Core user flow

  • User opens the app and signs in with email code authentication.
  • User syncs the card catalog and browses cards in Binder.
  • User marks owned copies, wanted cards, and custom card lists.
  • User creates and edits decks using official card data and deckbuilding helpers.
  • User can save deck snapshots, review versions, and log manual match results.
  • User can optionally manage friends and browse community deck activity.

What the app does not do

  • It does not automate Riftbound gameplay or enforce live game rules during play.
  • It does not act as a standalone playable digital client for Riftbound.
  • It does not resolve turns, attacks, effects, or hidden game-state interactions.
  • It is intended as a companion, organizer, and deckbuilding utility.
